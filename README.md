# ID3EA II Project Code

This is the code for our IDEA II Project written using MATLAB.

## Developing/Running this Project

Make sure to have the [Statistics and Machine Learning Toolbox](https://www.mathworks.com/products/statistics.html) installed to continue development of this project and/or run it.

### Installing the Statistics and Machine Learning Toolbox

To install the Statistics and Machine Learning Toolbox, click on the "Add-ons" button (the button with cubes), search for the package, click the install button, and follow the prompts.

## Running the Project

To run this project, run the ```main.m``` file.
